# نظام متجر بطاقات الشحن (Top-Up Cards Store)

## المميزات الرئيسية

### 1. إدارة الشركات والفئات
- إضافة شركات الاتصالات بشكل مرن (أوريدو، زين، آسيا سيل، إلخ)
- إضافة فئات الخدمات (ريمان، انترنت، إلخ)
- كل متجر له شركاته وفئاته الخاصة

### 2. المنتجات والأسعار
- إنشاء منتجات بطاقات شحن بفئات مختلفة
- دعم البيع بالمفرد (بطاقة واحدة)
- دعم البيع بالجملة (10 بطاقات، 50 بطاقة، 100 بطاقة بسعر مختلف)
- عرض عدد الأكواد المتاحة في لوحة تحكم التاجر

### 3. نظام الأكواد
- تخزين آمن للأكواد في قاعدة البيانات
- رفع الأكواد عبر ملف CSV
- حالات مختلفة للأكواد (متاح، مباع، منتهي الصلاحية)
- إخفاء الكود عن المشتري قبل عملية الشراء والدفع
- إظهار الكود فقط بعد تأكيد الطلب (عند الاستلام)

### 4. عملية الشراء
- اختيار الشركة والفئة والمبلغ
- اختيار نوع الشراء (مفرد أو جملة)
- عملية دفع سهلة وآمنة
- استلام الكود مباشرة بعد تأكيد الطلب

### 5. عدم التعارض مع المتاجر الموجودة
- نظام منفصل تماماً عن متاجر المنتجات العادية
- كل متجر يمكنه تفعيل متجر بطاقات الشحن اختيارياً
- الطلبات والأكواد منفصلة عن المنتجات العادية

## الجداول الجديدة

### topup_companies
```sql
- id: معرف الشركة
- store_id: متجر البيع
- name: اسم الشركة (أوريدو، زين، إلخ)
- logo_url: شعار الشركة
- is_active: حالة التفعيل
```

### topup_categories
```sql
- id: معرف الفئة
- store_id: متجر البيع
- name: اسم الفئة (ريمان، انترنت، إلخ)
- is_active: حالة التفعيل
```

### topup_products
```sql
- id: معرف المنتج
- store_id: متجر البيع
- company_id: الشركة
- category_id: الفئة
- amount: المبلغ (1000، 2500، إلخ)
- quantity_type: نوع الكمية (single, bulk_10, bulk_50, bulk_100)
- price: السعر
- bulk_price: سعر الجملة
```

### topup_codes
```sql
- id: معرف الكود
- store_id: متجر البيع
- topup_product_id: المنتج
- code: كود الشحن
- status: الحالة (available, sold, expired)
- assigned_order_id: الطلب المرتبط
```

## APIs المتاحة

### شركات الاتصالات
- `GET /api/topup/companies/:storeId` - جلب الشركات
- `POST /api/topup/companies` - إضافة شركة جديدة

### الفئات
- `GET /api/topup/categories/:storeId` - جلب الفئات
- `POST /api/topup/categories` - إضافة فئة جديدة

### المنتجات
- `GET /api/topup/products/:storeId` - جلب المنتجات مع عدد الأكواد المتاحة
- `POST /api/topup/products` - إنشاء منتج جديد

### الأكواد
- `POST /api/topup/upload-codes` - رفع أكواد من CSV
- `POST /api/topup/purchase` - شراء بطاقة (إخفاء الأكواد في الاستجابة)
- `GET /api/topup/order-codes/:orderId` - جلب أكواد الطلب (بعد الدفع فقط)

## خطوات الاستخدام

### للتاجر
1. التوجه إلى لوحة التحكم → متجر بطاقات الشحن
2. إضافة شركات الاتصالات (أوريدو، زين، إلخ)
3. إضافة فئات الخدمات (ريمان، انترنت)
4. إنشاء منتجات البطاقات
5. رفع الأكواد عبر ملف CSV
6. مراقبة المخزون والطلبات

### للزبون
1. التوجه إلى متجر بطاقات الشحن
2. اختيار الشركة والفئة والمبلغ
3. اختيار الكمية المطلوبة
4. إتمام عملية الشراء والدفع
5. استقبال كود الشحن بعد تأكيد الطلب

## ملاحظات الأمان
- الأكواد مشفرة وآمنة في قاعدة البيانات
- الكود لا يظهر للمشتري إلا بعد الدفع الفعلي
- كل كود يُستخدم مرة واحدة فقط
- تتبع كامل لمن اشترى كل كود
